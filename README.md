BrainGT
============
This repo is the official implementation of BrainGT: Learning Individual-to-Group Relationships with Graph Transformers Using Multimodal MRI for Brain Disease Classification
## Requirements:
* torch_geometric == 2.3.1
* pytorch == 1.13.1
* igraph == 0.10.8
* numpy == 1.21.5
* scipy == 1.7.3
